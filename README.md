# CSV Data Analyzer Web App

## Overview
This project is a simple web application built with Streamlit that allows users to upload a CSV file, preview the dataset, and visualize numeric columns using histograms.

## Features
- Upload CSV dataset
- Preview data instantly
- View column names
- Select numeric column for visualization
- Automatic histogram plotting

## Tech Stack
Python, Pandas, Matplotlib, Streamlit

## How to Run Locally
pip install streamlit pandas matplotlib
python -m streamlit run app.py
